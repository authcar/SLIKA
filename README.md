# 📸 SLIKA PHOTOBOOTH

**SLIKA PHOTOBOOTH** is a free **online photobooth website** (desktop only) that allows users to take pictures using their webcam, apply **filters**, choose **frames**, decorate with **stickers**, and download the final results as **JPG** or **GIF**.  
This is a fully personal project — designed and built **entirely from scratch**, including UI/UX and all front-end development.

---

## ✨ Features

- 📷 Capture photos using desktop webcam  
- 🎨 Apply various photo filters (pink, twilight, vintage, grayscale, dreamy, etc.)  
- 🖼 Select multiple frames for your picture  
- 🧸 Add cute and fun stickers  
- 🔄 Unlimited retakes  
- ⬇️ Download the result as **JPG** or **GIF**  
- 🚫 No installation needed — runs directly in the browser  

---

## 🛠 Tech Stack

- **HTML**, **CSS**, **JavaScript**  
- **Canvas API** for rendering, filters, frames, and exports  
- **Figma** for UI/UX design
